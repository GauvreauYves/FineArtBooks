Description:
"Rêveries" is a fine art photography book by Yves Gauvreau, printed using carbon pigment inks on archival Hahnemühle Photo Rag 188 gsm paper.  
This reference edition includes the complete book (PDF), technical notes (PDF), and structured metadata (Dublin Core) for institutional archiving, cataloging, and long-term access.
